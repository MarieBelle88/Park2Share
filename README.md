# Park2Share

**Park2Share** is an app that transforms airport parking (or more) into a car-sharing opportunity. Car owners can rent out their vehicles while they are traveling, offering travelers convenient access to a car.

## Features

- Rent out your vehicle while you're away.
- Book a vehicle from other travelers for your trip.
- Secure and insured transactions.
- User-friendly interface for easy car sharing and renting.

## Installation

1. Clone the repository:

   git clone https://github.com/your-username/park2share.git

2. Navigate to the project directory:

cd park2share

3.Install the required dependencies:

npm install

4. Start the app:

npm start
